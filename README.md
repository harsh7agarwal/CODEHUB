# GitHub Clone — Version Control Platform

Full-stack learning project: Express + MongoDB + React + AWS.

## Structure

- `backend/` — REST API (Node.js + Express)
- `frontend/` — React UI (added in a later phase)
- `cli/` — Yargs CLI (added in a later phase)

## Backend (Phase 0)

**Option A — from project root (recommended):**

```bash
cd c:\Users\bit\Desktop\MYGITHUBPRO
npm run dev
```

**Option B — from backend folder:**

```bash
cd backend
npm install
npm run dev
```

> Do not run `npm init` again at the root. The API lives in `backend/server.js`, not `index.js`.

Health check: [http://localhost:3000/health](http://localhost:3000/health) or [http://localhost:3000/api/health](http://localhost:3000/api/health)
